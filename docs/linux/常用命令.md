|ALT + F|forward|跳向下一个空格|
|ALT + B|backward|跳回上一个空格|

### 3.2 插入模式

指令|模式|描述
|--|--|--|
i|insert|在当前光标位置之前开始插入
a|-|在当前光标位置之后开始插入
I|-|在当前光标位置所在行开头处插入
A|-|在当前光标位置所在行结尾处插入
o|-|在当前行的下方新打开一个行插入
O|-|在当前行的上方新打开一个行插入
R|repalce|从光标所在位置的字符开始替换

注：处于替换模式时，不会插入文本，所输入的每个字符均将替换掉当前文档中的字符。
